Career Guidance System

An AI-powered platform designed to assist students and professionals in choosing the right career path based on education, interests, and job market trends.

  • Personalized career & university program recommendations
  • AI-driven career matching & industry insights
  • Real-time job market demand trends & salary projections
View Code Live Demo

Technologies Used

Flask, FastAPI (AI Integration)
React, Bootstrap, JavaScript
PostgreSQL for Structured Data
AI Model for Career Recommendations
API Integration (Job Market Insights)
AWS (S3, EC2), Cloudflare Security

📌 How It Works – Step by Step

Step 1: User Registration

Users create an account and enter their education background.

Step 2: AI Career Matching

System uses AI models to suggest careers based on data trends.

Step 3: Explore Career Insights

Users get salary data, job demand trends, and career growth paths.

🔹 Key Features

  • 🔍 Smart Career Suggestions – AI-powered recommendations.
  • 📊 Real-time Industry Trends – Job market insights & salary projections.
  • 🎓 Education Pathways – Best university programs based on subjects.
  • 🗂️ User Dashboard – Save career choices and track progress.
  • ⚡ Fast & Secure – Uses JWT authentication for secure login.
  • 📂 Admin Panel – Universities can manage career/course data.
  • 🌍 Multilingual Support – Available in English, French, and Spanish.
  • 🔗 API Integration – Connects to job boards and university databases.

Job Management System

A role-based job tracking system that streamlines job workflows, enhancing team efficiency, transparency, and accountability.

  • Role-Based Access Control – Users can only access their assigned jobs.
  • Real-Time Job Tracking – View live job statuses and progress updates.
  • Secure Authentication – Protects sensitive job data using JWT-based login.
View Code Live Demo

Technologies Used

Flask (Python), SQLAlchemy ORM
Bootstrap, JavaScript
PostgreSQL (Job & User Management)
JWT Authentication
Chart.js for Job Reports
AWS EC2, RDS for Hosting

📌 How It Works – Step by Step

Step 1: User Registration & Role Assignment

Admins approve new users and assign them roles (IT Tech, Designer, General User).

Step 2: Job Creation & Assignment

Users create jobs, set deadlines, and assign them to relevant team members.

Step 3: Real-Time Monitoring

Admins track job progress via graphical reports & status updates.

👥 User Roles & Permissions

  • 🛠️ Superuser – Manages users, assigns roles, generates reports.
  • 💻 IT Technician – Handles technical jobs, updates statuses.
  • 🎨 Designer – Creates and manages design-based jobs.
  • 📂 General User – Requests jobs, tracks assigned tasks.

🔹 Key Features

  • 🗂️ Role-Based Access – Ensures secure and restricted job data visibility.
  • 📊 Graphical Reports – Displays job stats with real-time charts.
  • ✅ Job Status Management – Jobs can be approved, rejected, or completed.
  • 🔒 Secure Authentication – Uses JWT-based login system.
  • 📅 Deadline Tracking – Ensures timely completion of tasks.
  • ⚡ Optimized Performance – Built for speed with Flask & PostgreSQL.

University Timetable Application

An AI-powered scheduling system designed to automate university class timetables, prevent conflicts, and optimize resource usage.

  • Prevents Class Conflicts – Eliminates overlapping course schedules.
  • Room & Resource Optimization – Ensures efficient use of university facilities.
  • Printable Timetables – Generate PDF schedules for students and faculty.
View Code Live Demo

Technologies Used

Flask (Python), SQLAlchemy ORM
Bootstrap, JavaScript
PostgreSQL (Course & Timetable Storage)
AI-Based Conflict Detection
PDF Export for Printable Timetables
AWS EC2, Firebase Authentication

📌 How It Works – Step by Step

Step 1: Course & Lecturer Input

Users enter course details, lecturer names, and time slots.

Step 2: AI-Based Conflict Detection

The system analyzes schedules and resolves conflicts automatically.

Step 3: Generate Printable Timetable

Students and faculty download their schedules as PDFs.

👥 User Roles & Permissions

  • 📚 Students – View and download their personalized schedules.
  • 🎓 Lecturers – Assign class time slots and request changes.
  • 🏫 Admins – Manage course scheduling and conflict resolution.
  • 📅 University Staff – Optimize schedules and track resource usage.

🔹 Key Features

  • ✅ AI-Based Conflict Resolution – No overlapping courses.
  • 🖥️ Fully Responsive – Works on mobile & desktop.
  • 📅 Optimized Room Scheduling – Prevents double bookings.
  • 📂 Printable PDFs – Students & lecturers can download their timetables.
  • 🔒 Secure Authentication – Firebase login ensures safe access.
  • ⚡ Fast & Scalable – Designed to handle large student populations.
